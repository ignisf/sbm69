# sbm69

This project provides a library and a CLI to export blood pressure measurements
as CSV from Hans Dinslage GmbH/Silvercrest SBM69 Bluetooth Blood Pressure
Monitor devices.

## Installation

Just clone the repo and then execute:

    $ pip install sbm69

## Usage

    $ sbm69 [-h] [address]

For example:

    $ sbm69
